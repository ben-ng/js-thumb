What Is This?

It's a library for thumbnailing images and videos on your users' browsers. This page demonstrates cross-domain thumbnailing. See the regular demo here.

How does it work?

HTML5 Canvas and Video tags. This means it won't work on all browsers. You can read more about it.

How does the cross-domain stuff work?

Browser support at this time is poor for the crossdomain attribute. Only Chrome Canary seems to be behaving as expected.

We'll create a tiny page on the source server that loads the video and makes it available for thumbnailing. Include this page in an iframe, and send the thumbnail to the parent page.
